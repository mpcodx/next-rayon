## 5. Performance SEO Recommendations

### Page Speed

- Optimize above-the-fold media and avoid oversized hero assets
- Use `next/image` with precise width and height values
- Compress all images to modern formats like WebP or AVIF
- Limit third-party scripts and defer anything non-essential
- Split heavy client components where SSR or static rendering is enough

### Core Web Vitals

- Keep LCP under 2.5 seconds by optimizing hero images, fonts, and server response
- Keep INP under 200ms by reducing main-thread work and avoiding large client bundles
- Keep CLS under 0.1 by reserving layout space for images, embeds, and dynamic sections

### Mobile Optimization

- Prioritize mobile-first spacing, heading sizes, and form usability
- Test tap targets and accordions on small screens
- Reduce motion and background effects on lower-powered devices
- Keep forms short and friction-light on mobile

### Image Optimization

- Add descriptive alt text for service and trust imagery
- Lazy-load below-the-fold images
- Serve responsive image sizes based on viewport
- Replace decorative oversized PNGs where SVG or compressed assets are enough

### Lazy Loading

- Lazy-load testimonial carousels, maps, and non-critical visual sections
- Keep primary content SSR or static for search visibility
- Avoid lazy-loading content that must be indexed immediately above the fold
